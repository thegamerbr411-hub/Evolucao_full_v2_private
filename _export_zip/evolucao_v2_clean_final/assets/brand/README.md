# Evolucao Brand Kit

Direcao visual: fitness + dados + progresso, estilo premium e limpo.

## Arquivos

- `logo-evolucao.svg`: logo principal horizontal.
- `splash-screen.svg`: arte base para splash.
- `../icons/app-icon.svg`: icone principal do app.
- `../icons/badge-pack.svg`: pacote de badges base (bronze/prata/elite).
- `../illustrations/challenge-banner.svg`: banner de desafios.
- `../illustrations/empty-state-pack.svg`: empty-state geral.
- `../illustrations/loading-motion.svg`: loading ilustrativo.

## Exportacao sugerida

- App icon: 1024x1024 PNG
- Splash: 1242x2688 PNG
- Banner de desafio: 1200x420 PNG
- Empty state: 1280x720 PNG

## Observacoes

- Assets proprietarios do projeto Evolucao Full V2.
- Nao usar marcas externas nem logotipos de terceiros.
