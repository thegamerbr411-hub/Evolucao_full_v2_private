DASHBOARD OPERACIONAL

Servidor:
- dashboard/server.js (API + static dashboard/public)
- boot com PORT por env
- log de boot: Server running on port + emoji log

Fila/worker:
- queue BullMQ em dashboard/src/server/queue.js
- filas: bugQueue, aiQueue, fixQueue, retestQueue
- retries: 3 tentativas
- backoff exponencial
- timeout por job
- log de erro estruturado

Workers:
- dashboard/worker.js
- dashboard/workers/aiWorker.js
- dashboard/workers/fixWorker.js
- dashboard/workers/retestWorker.js

UI web:
- dashboard/public/index.html
- cards de bugs, fila/worker, status IA/retest
- botoes por bug: Corrigir com IA, Re-testar, Aplicar decisao
