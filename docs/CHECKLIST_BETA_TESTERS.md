# CHECKLIST BETA TESTERS (10 pessoas)

## Entrada do tester
- [ ] instalacao do APK ok.
- [ ] primeiro acesso sem crash.
- [ ] conta vinculada (email ou Google).

## Fluxos obrigatorios
- [ ] home navegavel.
- [ ] treino iniciado e finalizado.
- [ ] nutricao registrada manualmente.
- [ ] OCR testado com imagem real.
- [ ] coach recebeu e respondeu.
- [ ] social com amigo ou desafio.
- [ ] perfil exportou pacote beta.

## Coleta de evidencia
- [ ] feedback em texto.
- [ ] print/video se houve falha.
- [ ] horario, device e versao do app.

## Classificacao de problema
- [ ] bloqueador
- [ ] alto
- [ ] medio
- [ ] baixo

## Encerramento do ciclo
- [ ] feedback consolidado em melhorias.
- [ ] timeline e observabilidade anexadas.
